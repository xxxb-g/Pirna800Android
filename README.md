# Pirna800Android
Eine schlichte kleine App, um [Pirna800](https://pirna800.de) auf dem Handey zu nutzen. 

### Testkonfigurationen

Diese App wurde auf unterschiedlichen Handys getestet:
- Galaxy A14 (magisk, Android 15)
- Galaxy S6 Edge (magisk, /e/OS)
- Galaxy S7 (/e/OS)
- Pixel 9a (emuliert, Android 16)
- Fairphone 3
- Android 6


Wie Sie Pirna800 unterstützen können, steht auf dieser [Internetseite](https://pirna800.de/baumspenden)
